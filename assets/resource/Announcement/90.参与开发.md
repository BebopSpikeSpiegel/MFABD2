# 参与开发

这个项目长期是我一人在填战壕，我**很欢迎友军**。如果你愿意一起做，门在这儿，开着。

> 完整的参与指南（要什么/不要什么、提交后会经历什么、怎么搭环境）在 GitHub：
> <a href="https://github.com/sunyink/MFABD2/blob/main/CONTRIBUTING.md" target="_blank">**CONTRIBUTING.md**</a>

---

## 一句话流程

Fork → 开个分支随便改 → 提 PR。

[color:gray]提到哪个分支不用纠结，在 PR 里说一句你做了什么，我帮你归位。拿不准就先开个 Issue 聊。[/color]

---

## 我大概会要什么

- 游戏更新后**坏掉的识别/流程**修复（新卡带、改版界面）——永远缺人。
- 新功能的 **Pipeline 节点**或 **Agent**（自定义识别/动作）。
- 新地图/活动的**资源、截图**。
- 文档、错别字——再小都欢迎。

[color:gray]方向性大改（动核心流程、跨资源包），做之前先开 Issue 对一下方向。[/color]
